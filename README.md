# Speech Therapy Website — React Project

This is a website built in React for a speech therapist. It includes informational sections, a blog with dynamically loaded articles, Calendly integration, and a GDPR-compliant cookie banner.

## 📦 Getting Started

Follow the steps below to run this project locally on your machine.

### 1. Clone the repository

```bash
git clone https://github.com/polaresweb3/mova-website
cd mova-website

2. Install dependencies
Make sure you have Node.js (version 18 or newer) installed.
npm install

3. Run the development server
npm start

The project will be available at:
📍 http://localhost:3000
